### Overview  
Central legal hub for the Hollow House Institute, including the Master License Suite and individual licenses (AHRL-1, RAP-DL 1.0, FBCR-1, 444-A, TCDPL-4.4).

### Structure  
- `src/` — Source code, scripts  
- `data/` — Datasets (raw, processed, samples)  
- `docs/` — Documentation, outlines, diagrams  
- `assets/` — Images, diagrams  
- `legal/` — License materials  
- `LICENSE.md` — License summary
