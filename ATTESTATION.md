# License Integrity & Attestation

All license instruments in this repository are subject to integrity verification.

## Hashing

Each license file SHALL be hashed using SHA-256 at the time of issuance.

Any modification invalidates prior attestations.

## Verification

Authorized recipients may request:
- File hash
- Issuance timestamp
- License version identifier

Verification requests:
data@hollowhouse.org

## Legal Effect

Tampering, modification, or redistribution of license text constitutes a breach of license and may trigger enforcement actions.
